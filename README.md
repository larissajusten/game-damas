# Damas

Damas is a game where 2 people have to make moves to "eat" as many of the opponent's pieces as possible.

## Project Structure

```json
|--src
  |--damas.css
  |--damas.html
  |--damas.js
  |--assets
    |--scss
      |--colors.scss
      |--fonts.scss
      |--variables.scss
    |--objects
```

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install Damas.

```bash
npm install
```

## Usage

```bash
npm start
```
